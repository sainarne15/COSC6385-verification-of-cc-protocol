## How to run the code:
1. cd Murphi3.1/src
2. make mu   (make clean if already make was run)
3. make install
<br>**go to the Murphi3.1/bin**
 In Murphi3.1/bin, create a symbolic link
 4. ln -sf mu. mu
 5. cd ..
 6. cd ex
 7. ../bin/mu toy/pingpong.m
 8. Inisde the ex/toy,   make pingpong
 9. ./pingpong -v
 10. ./pingpong -h
